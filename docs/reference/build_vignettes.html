Build vignettes section

Description

build_vignettes() renders R Markdown files from vignettes/ to vignettes/

Arguments

pkgPath to package
lazyWhether to skip files that haven't changed
quietWhether to suppress messages
previewWhether to preview after building using a local server (requires the servr package)

Return

Invisible TRUE if the vignettes were built successfully

Examples

build_vignettes()

# Rebuild only if source is newer than destination
build_vignettes(lazy = TRUE)

# Hide build messages
build_vignettes(quiet = TRUE)

# Preview the vignettes index after building
build_vignettes(preview = TRUE)
