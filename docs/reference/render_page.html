Render page with minimal template

Usage

render_page(pkg = ".", name, data, path, quiet = FALSE)
data_template(pkg = ".", root_path = "")

Arguments

pkgPath to package to document.
nameName of the template (e.g. "home", "reference-topic")
dataData for the template.
pathLocation to create file; relative to destination directory.
quietIf quiet, will suppress output messages
root_pathRelative path to site root (automatically calculated)

Description

Each page uses simple templates: "head", "header", "content", and "footer".
