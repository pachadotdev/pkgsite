Build news page

Description

Generates a news page from NEWS.md

Arguments

pkgPath to package to document.
quietIf quiet, will suppress output messages
previewIf TRUE, the news page will be opened using a local server (requires the servr package)

Return

Invisible TRUE if the news page was built successfully

Examples

build_news()

# Hide build messages
build_news(quiet = TRUE)

# Preview the news page after building
build_news(preview = TRUE)
